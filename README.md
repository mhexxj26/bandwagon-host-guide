## 搬瓦工为啥叫搬瓦工：一个谐音故事

原因说穿了特别简单——就是发音像。

BandwagonHost，国内用户按发音拆开念：Ban·Wa·Gon，读起来就很像"搬瓦工"。久而久之，论坛帖子里大家图省事，直接写"搬瓦工"，反而比原名传得更广。现在这个外号已经比它的英文名更出名了，新人第一次接触基本都是从"搬瓦工"这个词进来的，反过来再去查 BandwagonHost 是什么。

顺便说一下，Bandwagon 这个英文词本身挺有意思。它的原意是游行庆典里的花车、彩车——那种站在队伍最前面、又大又漂亮的主角车，有"引领风潮"的意思。所以 BandwagonHost 这个名字原来带着点"跟上潮流"的寓意，只不过到了中文语境里，彻底变成了一个干活的形象。

## 线路怎么看：CN2 GT 和 CN2 GIA 到底差在哪

这是买搬瓦工绕不开的一个问题。

**CN2 GT**：中国电信的 CN2 线路，回程走 CN2 骨干，但在流量高峰时期会有一定拥堵，晚上波动明显。价格便宜，适合对速度要求不高的基础使用。

**CN2 GIA**：CN2 的高端版本。去程和回程都走 CN2 骨干，专用带宽，拥堵控制更好，晚高峰稳定性比 CN2 GT 强得多。三网优化的意思是电信、移动、联通三家运营商都走了对应的优化路由。延迟低、丢包率低。

讲真，如果你只是想跑个脚本、偶尔测个速，KVM 套餐的 CN2 GT 够用了。但如果是建站，尤其是面向国内访客的网站，CN2 GIA 在晚高峰的表现差距很明显——加载速度差个 1-2 秒，跳出率差距不小。
